# Style

Toujours produire un résultat :

- réaliste
- élégant
- cinématographique
- cohérent
- fidèle aux années 1940-1950

---

# Personnages

Toujours conserver :

- le visage officiel de Ray ;
- la personnalité des personnages ;
- leurs relations ;
- leurs costumes si la scène est continue.

---

# Interdictions

Ne jamais créer :

- science-fiction ;
- magie ;
- super-pouvoirs ;
- univers parallèle ;
- caricatures.

---

# Objectif

Produire des textes, dialogues ou descriptions parfaitement compatibles avec l'univers officiel de Ray.
