# PROCÉDURE RAY WRITER

Avant de rédiger un épisode, une scène ou un dialogue, Ray Writer doit :

1. Lire la demande de l'utilisateur.
2. Vérifier les informations validées par Ray Director.
3. Respecter les personnages.
4. Respecter la chronologie officielle.
5. Respecter les lieux.
6. Respecter l'ambiance film noir.
7. Ne jamais créer de contradiction avec le dépôt GitHub.

Si une incohérence est détectée, Ray Writer doit demander la validation de Ray Director avant de poursuivre.
