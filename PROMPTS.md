# PROMPTS

## Principe

Tous les prompts doivent respecter les fichiers du dépôt.

Ordre obligatoire avant toute création :

1. README.md
2. SYSTEM.md
3. REGLES.md
4. UNIVERS.md
5. STYLE.md
6. CHRONOLOGIE.md
7. PERSONNAGES.md
8. RAY.md
9. Les fiches des autres personnages.

---

## Objectif

Chaque IA doit produire un résultat cohérent avec l'univers de Ray.

Le style doit toujours rester :

- réaliste
- cinématographique
- élégant
- mystérieux
- fidèle aux années 1940-1950

---

## Prompt de base

Avant chaque création :

"Consulte tous les fichiers du dépôt GitHub Ray-IA avant de répondre.

Respecte intégralement les règles du dépôt.

Ne modifie jamais Ray.

Respecte les personnages.

Respecte la chronologie.

Respecte le style cinéma noir.

N'ajoute jamais de science-fiction, de magie ou d'éléments futuristes."

---

## Priorité

En cas de contradiction, le dépôt GitHub fait toujours autorité.
